# Журнал изменений

## 1.1.4 — 06.03.2026

### Мониторинг
- Для `HTTP 503` (если код не входит в разрешенные) добавлен статус `ISSUE` (оранжевый) вместо мгновенного `DOWN`.
- Добавлена ускоренная эскалация для `HTTP 503`: если сервис не восстановился в течение 2 минут после последнего `UP`, статус автоматически переводится в `DOWN`.
- Добавлены автотесты на поведение `503`:
  - `ISSUE` при первичной фиксации ошибки;
  - переход в `DOWN` после порога эскалации.
