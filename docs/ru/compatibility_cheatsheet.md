# Compat — шпаргалка

## Быстрый алгоритм
1) После логина откройте мастер совместимости (появится автоматически, если есть проблемы).
2) Для вкладки со статусом:
   - `needs_attention` → сначала **Partial adapt**
   - `needs_reinit` → нужен **Full reset** (деструктивно)
   - `broken` → проверьте логи/БД, затем принимайте решение
3) При Full reset: **сделайте backup** перед действием.
4) Запустите job и дождитесь `finished` (прогресс 100%).

## Partial adapt vs Full reset
- Partial adapt: безопасно, бизнес-данные сохраняются.
- Full reset: удаление данных модуля + восстановление дефолтов.

## Где смотреть прогресс
- Мастер показывает прогресс автоматически (poll).
- API:
  - `GET /api/app/jobs/:id`
